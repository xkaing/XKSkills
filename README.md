# XKSkills

个人收藏的 Agent Skills 集合。

## 项目结构

```
XKSkills/
├── skills/
│   └── *.md            # Skill 定义文件 (Markdown + Frontmatter)
├── README.md           # 项目说明
└── LICENSE.txt         # 许可证
```

## Skill 格式

每个 Skill 使用 Markdown 格式定义，顶部使用 YAML Frontmatter，包含以下字段：

- `name`: Skill 名称
- `description`: 简短描述
- `instruction`: 详细的指导指令（支持 Markdown）
- `tags`: 相关标签
- `version`: 版本号
- `author`: 作者
- `license`: 许可证
